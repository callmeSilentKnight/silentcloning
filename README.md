# Install
```
$ pkg update && pkg upgrade
$ pkg install git
$ pkg install python2
$ pip2 install requests
$ pip2 install mechanize
$ git clone https://github.com/callmeSilentKnight/silentcloning
$ cd silentcloning
$ python2 silentcloning.py
```
